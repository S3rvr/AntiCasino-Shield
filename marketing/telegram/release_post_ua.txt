🚀 AntiCasino Shield v1.1.1 — MVP-реліз

Перша публічна версія AntiCasino Shield готова для тестування.

Що нового:
- Windows-додаток для блокування казино та букмекерських сайтів;
- інсталятор;
- перевірка прав адміністратора;
- резервна копія та відновлення hosts;
- сильний захист із паролем і затримкою;
- журнал дій та crash-log;
- повний reset захисту;
- сторінка підтримки проєкту без реклами.

Як встановити:
1. Завантажте AntiCasinoShield_Setup_v1.1.1.exe.
2. Запустіть інсталятор.
3. Дозвольте права адміністратора.
4. Відкрийте програму та натисніть “Увімкнути захист”.

⚠️ Windows SmartScreen може показати попередження, бо це новий unsigned exe без цифрового підпису. Це не означає автоматично, що файл небезпечний. Завантажуйте програму тільки з офіційної сторінки або GitHub release.

AntiCasino Shield — не медична чи психологічна допомога, а практичний інструмент самоконтролю.
