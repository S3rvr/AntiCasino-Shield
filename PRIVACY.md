# Privacy Policy

AntiCasino Shield is a local Windows self-control tool.

## Data Collection

AntiCasino Shield does not collect personal data, does not track browsing history, and does not send telemetry to any server.

## Local Files

The app stores configuration, custom domains, and logs locally on the computer where it is installed. These files are used only to keep the app working and to help users diagnose issues.

## System Changes

When protection is enabled, AntiCasino Shield can update the local Windows `hosts` file, add browser URLBlocklist policies, flush the local DNS cache, and optionally create a Windows startup task. These changes happen locally.

## Removal

Users can disable protection from the app, run `reset_system_admin.bat`, or run `uninstall.bat` from the installation folder. Reset removes AntiCasino Shield managed hosts entries, browser URLBlocklist entries, and autostart configuration.
